- [ ] Pirma c# skripta izveide
- [ ] Audio source ieviesana
- [ ] Scrollview elementa lietojums
- [ ] Otras ainas (scene) ieviesana
- [ ] Collider 2D un Rigidbody 2D (fizikas speka iedarbiba)
- [ ] Virtulu spavneris
